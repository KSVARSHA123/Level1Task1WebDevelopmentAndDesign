Portfolio


Project Title: Portfolio Website


Project Description:


Create a sleek personal portfolio website using HTML and CSS to showcase skills and projects. The website features sections like Home, About Me, Portfolio, and Contact. It's designed to provide an overview of the creator's background, display projects with descriptions and images, and facilitate inquiries. The site is mobile-responsive and optimized for search engines. Technologies used: HTML, CSS, and optionally JavaScript for interactivity.


Outcome:

A professional and user-friendly portfolio platform for self-promotion and skill demonstration.



Project Title: Stock Market Tracker


Project Description:


Developed a Stock Market Tracker web application using HTML, CSS, and JavaScript. The app allows users to search and monitor real-time stock data by entering company symbols. It retrieves stock information from an API, displays stock prices, charts, and provides historical data. Users can save their favorite stocks and track them conveniently. The project involved responsive design for mobile and desktop. It's a valuable tool for investors and traders to stay updated on stock market trends.


Outcome:

A user-friendly and informative stock tracker application offering real-time data, charts, and customization for tracking personal investment portfolios.
